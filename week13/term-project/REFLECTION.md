# Week 13 Reflection

Week 13 focused on building a modular React Dashboard that extends the marketing page with real tourism data and live weather insights. The dashboard consists of three Chart.js visualizations: visitor arrivals and hotel occupancy percentages from DBEDT (Department of Business, Economic Development and Tourism) Hawaiian tourism data, and a live six-hour weather forecast from OpenWeatherMap. The component architecture prioritizes modularity—the Dashboard is self-contained with its own weather fetching logic, separate from the MarketingPage component, so it can be wrapped with authentication middleware in Week 14 without requiring refactoring. Both components receive data through props and include graceful fallback strategies: when APIs are unavailable or keys are missing, the page displays demo data and status messages instead of crashing. In Weeks 14–15, the plan is to add Passport.js authentication protecting an `/admin/dashboard` route, integrate a local foods API to surface nearby restaurants aligned with the Australian visitor segment, and embed a live surf cam feed to reinforce the property's position as a surfer's destination.
