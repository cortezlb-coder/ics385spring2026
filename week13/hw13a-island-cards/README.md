# HW13-A Island Cards

A small Vite + React project for ICS 385 Week 13. It uses props, `Array.map()`, `Array.filter()`, and `Array.reduce()` to render a filtered island card grid.

## Run

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Files

- `src/App.jsx` holds the island data array and page layout.
- `src/IslandList.jsx` filters cards by visitor segment and calculates the average stay.
- `src/IslandCard.jsx` renders one island card with destructured props.
- `src/App.css` contains the assignment styling.
