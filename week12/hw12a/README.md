# Hawaii Island Cards (HW12A)

I built this React app with Vite for HW12A to practice JSX, functional components, props, and imports/exports by showing three Hawaii island cards (Maui, Oahu, and Kauai) with a short description and visitor tip for each one. I used an islands array in `App.jsx` and rendered the cards with `.map()` into a reusable `IslandCard` component, then added basic CSS so it is not plain/unstyled. I used GitHub Copilot (GPT-5.3-Codex) to help draft and organize parts of the code, and I still tested and adjusted the final version myself.
